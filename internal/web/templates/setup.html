What this wizard does

The setup wizard tracks five onboarding steps and points you at the right form for whichever one comes next. It does not perform any actions on its own — every step links to the same dedicated form the navigation bar exposes, so power users can ignore the wizard entirely. State (done / skipped) is derived from the filesystem each time you load this page, so refreshing always shows the current truth.
